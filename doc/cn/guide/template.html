Blade

模版引擎

blade默认提供了JSP渲染，也支持其他模版引擎的加入， 现在已经完成了几款我认为比较简洁优秀的模版引擎插件：

  • 基于jetbrick-template的模版引擎blade-jetbrick
  • 基于velocity的模版引擎blade-velocity
  • 基于beetl的模版引擎blade-beetl

使用起来都很简单，在应用初始化的App类里加入 Blade.viewEngin(new JetbrickRender()); 即可，更多配置查看插件的文档，如果你使用JSP不用配置它即可。

下面是一个例子：

// 设置模板引擎
JetbrickRender jetbrickRender = new JetbrickRender();

jetEngine = jetbrickRender.getJetEngine();

Blade.viewEngin(jetbrickRender);

// 框架初始化后的操作
@Override
public void contextInitialized(ServletContext servletContext) {

    Map<String, Object> userProfile = userService.getProfile(1);

    JetGlobalContext globalContext = jetEngine.getGlobalContext();

    Constant.GLOBAL_CTX = globalContext;

    // 加入一个全局变量值
    Constant.GLOBAL_CTX.set(Map.class, Constant.USER_PROFILE_INFO, userProfile);

}
