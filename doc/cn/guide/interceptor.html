拦截器

这里的拦截器只是在Route请求前后进行过滤，实现也是在Route前后调用； 并非动态代理方式拦截器，基于路由而非方法。

使用：


@Interceptor
public class BaseInterceptor {

    TimerMonitor monitor = TimerManager.getTimerMonitor();

    @Before("/*")
    public void before(Request request, Response response){

        System.out.println("before...");
    }

    @After("/*")
    public void after(){
        System.out.println("after...");
    }
}
  • 用@Interceptor注解标识一个拦截器
  • @Before是路由的前置事件
  • @After是路由的后置事件

拦截器配置和路由配置类似
