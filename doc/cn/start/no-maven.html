Blade

不使用maven

有些同学可能还不会使用maven构建项目，建议有时间的话学习一下。

不使用maven构建基于blade的项目也非常简单，只要把你需要的jar包加进去就可以了。

blade核心库文件有2个

blade最新版

blade-kit最新版

需要其他相关的jar文件也可以在mvnrepository.com上面找到。

相信有同学已经迫不及待看看代码是怎么实现的，这里我把步骤写下

  1. 加入blade相关jar 加入最新版的blade.jar和blade-kit.jar即可

  2. 在xml中配置blade，这里的com.hello.App是你的全局配置类

     <web-app>
   <filter>
         <filter-name>BladeFilter</filter-name>
         <filter-class>blade.BladeFilter</filter-class>
         <init-param>
             <param-name>applicationClass</param-name>
             <param-value>com.hello.App</param-value>
         </init-param>
     </filter>
     <filter-mapping>
         <filter-name>BladeFilter</filter-name>
         <url-pattern>/*</url-pattern>
     </filter-mapping>
 </web-app>

  3. 在java代码中配置路由所在位置，并编写一个hello world

     /**
  * 全局配置
  *
  */
 public class App implements BladeApplication {

     @Override
     public void init() {
         Blade.routes("com.hello.route");
     }

 }

     @Path
 public class Hello {

     @Route("hello")
     public String hello(){
         System.out.println("hello blade");
         return "hello";
     }
 }

    最后别忘了在WEB-INF目录下写一个hello.jsp文件哦！
